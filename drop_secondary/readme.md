# 中文說明
死亡時掉落第二把武器

* 原理
    * 死亡時掉落手上裝備的第二把武器，譬如手槍、近戰武器、麥格農手槍
    * 可掉雙手槍

* 已知問題
    * 有時候死亡時沒有掉落近戰武器，原因是近戰武器被刪除了，問就是Valve魔法
